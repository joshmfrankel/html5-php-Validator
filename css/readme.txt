
CSS README


styleguide.css  -   Contains all base styles
                -   h1-h6, p, form, etc...

layout.css      -   Contains all design layout styles
                -   All container widths and positioning

effects.css     -   All reusable classes for effects
                -   Drop shadow, opacity, rounded

variables.css   -   Default SCSS variables 
                -   # of columns, width, rounded px for effects.css
